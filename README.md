## üì¶ Instalasi

### Metode 1: Download Release
1. Download file `.mcaddon` dari [Releases](https://github.com/Alifwag/addons-clearlagg/releases)
2. Buka file dengan Minecraft
3. Aktifkan di world settings

### Metode 2: Manual Install
1. Clone repository ini:
```bash
git clone https://github.com/Alifwag/addons-clearlagg.git
```

2. Copy folder **BP** ke:
```
%localappdata%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\behavior_packs\
```

3. Copy folder **RP** ke:
```
%localappdata%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\resource_packs\
```

4. Aktifkan kedua pack di world settings

## üéµ Cara Menggunakan Sistem Musik

### Menambah Musik dari Spotify

1. **Buka Menu Musik**
   ```
   !music
   ```

2. **Pilih "Tambah Musik Spotify"**

3. **Masukkan Data:**
   - **Link Spotify**: `https://open.spotify.com/track/...`
   - **Judul**: Nama lagu
   - **Artis**: Nama penyanyi

4. **Music Cassette Otomatis Dibuat!**
   - Item music disc muncul di inventory
   - Klik kanan untuk play musik
   - Metadata lengkap tersimpan

### Mengelola Koleksi Musik

```
!music ‚Üí Koleksi Musik Saya
```

Lihat semua musik yang telah ditambahkan dengan info lengkap:
- Judul & Artis
- Link Spotify
- Ditambahkan oleh siapa

### Menghapus Musik

```
!music ‚Üí Hapus Musik
```

Pilih musik yang ingin dihapus dari library.

## ‚öôÔ∏è Konfigurasi

### Default Settings

```javascript
{
  clearDelay: 300,        // 5 menit
  warningTime: 30,        // 30 detik
  language: 'id_ID',      // Indonesia
  enableAutoClean: true,  // Auto clean aktif
  cleanupTypes: {
    items: true,          // Bersihkan items
    arrows: true,         // Bersihkan arrows
    mobs: false,          // Jangan bersihkan mobs
    xp: false            // Jangan bersihkan XP
  }
}
```

### Mengubah Pengaturan

1. **Via Command**
   ```
   !settings
   ```

2. **Via Buku**
   - Klik kanan Settings Book
   - Atur semua opsi
   - Save otomatis

3. **Opsi yang Bisa Diatur:**
   - Delay pembersihan (30-600 detik)
   - Warning time (10-60 detik)
   - Bahasa (ID/EN)
   - Auto clean on/off
   - Tipe pembersihan (items, arrows, mobs, XP)

## üìä Performance Tips

### Optimasi Server

1. **Atur Delay Sesuai Kebutuhan**
   - Server kecil: 300-600 detik
   - Server besar: 120-300 detik

2. **Aktifkan Auto Clean**
   - Mencegah lag dari terlalu banyak entities
   - Force clean saat entity > 2000

3. **Pilih Cleanup Type**
   - Items & Arrows: Always recommended
   - Mobs: Hanya jika lag parah
   - XP: Jangan, valuable resource

4. **Monitor Stats**
   - Gunakan `!stats` berkala
   - Cek TPS dengan `!tps`
   - Watch entity count
